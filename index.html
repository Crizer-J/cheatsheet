Position and Property Values

Position Values

Value Example Code Meaning
static navbar { postion : static; } Position of the element is left in the normal flow of the document.
relative navbar { postion : relative; } Same as static until offset is established. The offset properties can be used to determine the actual position of the element relative to its original position.
absolute navbar { postion : absolute; } Enables an element to ignore sibling elements and instead be positioned relative to its closest parent element that is positioned with relative or absolute. Top, left, bottom and right, an element can be positioned anywhere as expected.
fixed navbar { postion : fixed; } The fixed element stays the same regardless of scrolling.

Display Values

Value Example Code Meaning
block .container1 { display: block; } Take up the entire width of their container. Height and Width can be manually adjusted.
inline .container1 { display: inline; } Take up as little space as possible. Can NOT have their height and width manually adjusted.
inline-block .container1 { display: inline-block; } Elements can appear next to each other. Can have their height and width manually adjusted.
